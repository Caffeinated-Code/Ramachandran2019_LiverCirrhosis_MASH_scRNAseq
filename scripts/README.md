# Utility Scripts

Supporting utilities live here. Core Seurat analysis stages live in `workflow/`.

| Script | Purpose |
|---|---|
| `create_demo_dataset.R` | Create the small GSE136103-derived demo dataset |
| `prepare_validation_datasets.py` | Stream and summarize validation datasets |
| `enrich_target_evidence.py` | Query public target evidence resources |
| `validate_repo_structure.py` | Lightweight repository integrity check |

These scripts are intended to be deterministic and safe to rerun.
