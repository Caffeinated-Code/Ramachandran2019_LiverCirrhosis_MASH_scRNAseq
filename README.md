# Ramachandran2019 Liver Cirrhosis MASH scRNA-seq

This repository is being built as a reproducible mini-pipeline for the Karyon Bio candidate assignment on single-cell transcriptomic discovery and prioritization of cell-type-specific biomarkers and therapeutic targets in human liver fibrosis.

The project will center on GSE136103 as the primary discovery dataset, with a scalable workflow for QC, annotation, differential expression, pathway analysis, biomarker prioritization, validation, reporting, and interactive visualization.
